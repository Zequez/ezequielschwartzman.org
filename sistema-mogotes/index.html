Sistema Cooperativo Barrio

Un [Mundo de Juego]

  1. El propósito es fomentar la Cooperación, la [Colaboración Creativa], las Posibilidades y el Empoderamiento entre los Participantes de un organismo social que esté anhelando más cohesión y trabajo en equipo entre sus humanos.
  2. Elegir un organismo social poco definido y poco centralizado, por ejemplo, un barrio, o una constelación de proveedores de alimentos agroecológicos en una bioregión.
  3. Utilizamos el Entorno Web como herramienta para crear el sistema, con el sustrato Spacious Substrate
  4. El [Sostenedor de Espacio] para el Mundo de Juego debe tener acceso a una computadora donde el Entorno Web esté preparado para usarse. Y debe saber como configurar el sistema para proyectarse a la web en un dominio en particular.
  5. Se elige un dominio web donde va a estar la página colaborativa.
  6. Se prepara
